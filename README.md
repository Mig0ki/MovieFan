# MovieFan
An Android application written in Kotlin that allows users to browse upcoming movies, add them to a "To-Watch Movies" list, 
and randomly select movies with the option to filter by genre. The app also lets users set notifications for movie release dates.
For proper functionality, the app requires adding your own API KEY from https://developer.themoviedb.org/docs/getting-started in file:
MovieFan -> app -> src -> main -> java/com/example/moviefan -> api -> models -> RetrofitClient.kt.
